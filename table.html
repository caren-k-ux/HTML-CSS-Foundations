HTML Tables

HTML tables allow web developers to arrange data into rows and columns

Define an HTML Table

A HTML table consists of table cells inside rows and columns

Each table cell is defined by a td tag.

td stands for table data

Everything between the td tag is the content of the table cell

Table Rows

Each table row starts with a tr tag and ends with a /tr tag.

tr stands for table row.

You can have as many rows as you like in a table; just make sure that the number of cells is the same in each row.

Table Headers.

Sometimes you want your cells to be the table header cells. In those cases us the th tag instead of the td tag.

th stands for table header.

Bu default th elements are bold and centered, but you can change that with CSS.
