Every webpage should have a page title to describe the meaning of the webpage

The title should describe the content and meaning of the webpage
The title is very important for search engine optimizaion. The text is used by search engine algorithims to decide the order when listing pages in search results.
The title element:
  • defines a title in the browser toolbar
  • provides a title for the page when it is added to favorites
  • displays a title for the page search engine resulsts
so,try to make the title as accurate and meaningful as possible.
