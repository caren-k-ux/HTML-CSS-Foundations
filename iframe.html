HTML Iframe Syntax

The HTML iframe tag specifies an inline frame.

An inline frame is used to embed another documents within the current HTML document.

Syntax:

< iframe src="url" title="description"></iframe>

Tip: It is good practice to always include a title attribute for the <iframe>. This is used by screen readers to read out what the content of the iframe is.

Iframe - Remove the Border

By default, an iframe has a border around it.

To remove the border, add the style attribute and use the CSS border property:
i.e: "border: none;"

With CSS, you can also change the size, style and color the iframes border:
i.e: "border: 2px solid red; "

Iframe - Target for a Link

An iframe can be used as the target frame for a link.
